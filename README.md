multi-page design (Home, About, Services, Portfolio, Industries, Contact).

## Getting started

1. Install [Quarto](https://quarto.org/).
2. Copy this project into your GitHub Pages repo, e.g.:

   ```bash
   git clone https://github.com/edwardarchaeology/edwardarchaeology.github.io
   cd edwardarchaeology.github.io
   # copy files from this folder into the repo
   ```

3. Render locally:

   ```bash
   quarto render
   quarto preview
   ```

4. Configure GitHub Pages to serve from the `docs/` directory on your
   main branch.

Images live in `assets/img/` and styles in `assets/scripts/site.css`.
